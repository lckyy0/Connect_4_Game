# Connect 4: Drop · Connect · Conquer

A clean, responsive, and playful web-based version of the classic Connect 4 game. Built entirely with Vanilla JavaScript, HTML, and CSS, this project focuses on a premium user experience without the need for heavy frameworks.

##  Features

* **Physical Toy Aesthetic:** Utilizes CSS Claymorphism and soft shadows to mimic the tactile feel of a real physical board game.
* **Satisfying Animations:** Custom CSS keyframes create a realistic "bounce" effect when chips drop into the board, alongside a fun confetti explosion for the winner.
* **Light & Dark Mode:** Fully integrated theme toggle that automatically saves your preference for your next visit.
* **Smart Responsive Layout:** * On desktop, the game features a collapsible sidebar that perfectly centers the board when folded away. 
  * On mobile, the UI seamlessly transforms into a compact top-bar layout without breaking the game's proportions.
* **No-Scroll Gameplay:** The layout is engineered to fit perfectly within a single screen (`100vh`), keeping the focus entirely on the game.

##  How to Play

Since this game is built with pure frontend technologies, there is no installation or build step required!

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. Enter your player names, pick your theme, and click **Start Game**.

##  Tech Stack

* **HTML5:** Semantic structure and accessible ARIA labels.
* **CSS3:** Flexbox, CSS Grid, custom properties (variables), and keyframe animations.
* **JavaScript (ES6+):** Clean, vanilla JS handling game logic, win/draw detection, and DOM manipulation.
